## Getting started

- Clone repository

```
git clone https://github.com/ThaoLyyy/javascript-training
```

- Checkout branch `develop`

```
git checkout develop
```

- Change directory to `big-practice` folder

```
cd big_practice
```

- Open cmd and install package

```
npm install
```

- Run application

```
npm start
```

- Run server

```
npm run dev
```

- Open browser with the link below

```
http://localhost:1234
```
